## 系统运行级别

|运行级别|含义|
|-|-|
|0|关机|
|1|单用户模式，主要用于系统修复|
|2|不完全的命令行模式|
|3|完全的命令行模式，就是标准字符界面|
|4|系统保留|
|5|图形模式|
|6|重启动|

~~~
runlevel            查看系统的运行级别
init 运行级别        改变当前的运行级别

例：查看系统当前运行级别
runlevel
N 3

会显示2个数字，前一个数字代表系统前一个运行级别，后一个数字代表当前的运行级别，N表示没有前一个运行级别

例：改变当前运行级别
init 0
相当于关机，但是不建议这么使用，容易造成数据丢失，推荐使用shutdown
~~~

### 系统默认运行级别




### /etc/rc.d/rc.local

系统启动时执行一次，相当于开机启动文件，需要开机自启得程序可以放这里

```sh
[root@localhost ~]# ll /etc/rc.local 
lrwxrwxrwx. 1 root root 13 9月  11 2019 /etc/rc.local -> rc.d/rc.local
```
