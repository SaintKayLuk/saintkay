## intel

命名规则例子：
```
Intel Core i5-8600K

Intel   品牌，Intel和AMD
Core    系列，酷睿系列core 或 至强Xeon
i5      等级，分为i3,i5,i7,i9
8600    第一个数字代表代数，8就代表第八代酷睿
        600代表SKU，数字越大，频率越高，性能越好

后缀    分为桌面版和笔记本版，有区别
    桌面版    
        K 不锁频，可超频使用
        F 不带核显，不带f后缀就是带核显
        X 至高性能,高端产品，HEDT平台CPU
        E 最高级，和X同时出现，只出现在i9
    笔记本
        U 低电压版，一般用于轻薄本
        H 标准电压，不可拆卸
        G 和AMD合作款cpu，CPU由intel生成，GPU由AMD生产

```
