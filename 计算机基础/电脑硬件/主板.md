# 版型
* E-ATX：   高性能主板，芯片组都是X字母开头，适合使用带X后缀的处理器，价格高
* ATX ：    一般大小为 305 x 244 mm，俗称大版，扩展性能好，接口全
* M-ATX：   一般大小为 244 x 244 mm，正方形的，比较好辨认，俗称小板
* ITX：     一般大小为 170 x 170 mm，小正方形，比 M-ATX 更小

## 大小
```
E-ATX > ATX > M-ATX > ITX
```

# 芯片组等级

Intel 和 AMD 处理器所用主板是不同的，也不兼容，需要根据处理器来选择主板

## Intel

Intel主板芯片组有四个等级 X > Z > B > H
同等级看第二位数字，数字越高，越高端

### X字母开头
最高级，用来搭配高端cpu，一般cpu型号后缀带有 X 字母
比如 X299 主板，可以搭配 i9-7960X 等

### Z字母开头 
次高端，Z字母开头的主板都支持超频，搭配的cpu一般带有 K 字母后缀
比如 Z370 主板，可以搭配 i5-8600K 等

### B字母开头
中端，这种主板不支持超频，主要搭配不带 K 字母后缀的cpu
比如 B365 主板，就能使用 i5-9400 或 i5-9600F 等

### H字母开头
入门级，不支持超频

## AMD

AMD主板芯片组有3个等级 X > B > A
同等级看第二位数字，和Intel一样，第二位数字越大越高端

### X字母开头
最高级，支持自适应动态扩频超频，和Intel一样，也是搭配AMD中带 X 字母后缀的处理器

### B字母开头
中端主流，支持超频，不支持完整的自适应动态扩频超频

### A字母开头
入门级，不支持超频
